# 🏠 AI Smart Home Energy Monitor

An AI-powered smart home energy monitoring system that simulates
real household appliance data, detects anomalies, and predicts
future energy consumption — built entirely through simulation
and machine learning, no hardware required.

## 🎯 Features
- 📡 Simulates 5 home appliances (Fan, AC, Light, Fridge, Washing Machine)
- ⚡ Generates 720 hours of realistic energy consumption data
- 🤖 AI Anomaly Detection using Isolation Forest algorithm
- 🔮 24-hour Energy Prediction using Linear Regression
- 📊 Beautiful visual dashboard with real-time graphs

## 🛠️ Technologies Used
- Python 3.13
- NumPy and Pandas for data simulation
- Scikit-learn for machine learning models
- Matplotlib and Seaborn for visualization

## 🚀 How to Run
- python simulator.py
- python train_model.py
- python dashboard.py
